Piazza courses to LinkedIn profile

Instructions

Drag the link below to your bookmarks bar to use it:

")[1].split(" Piazza to LinkedIn


Go to piazza.com and click the bookmarklet.
A dialog should pop up with the input highlighted (this is a JSON list of your courses and their numbers). Use Ctrl+C (Cmd+C) to copy it.

Go to linkedin.com/profile/edit.
Next, navigate to the Courses section that allows you to add courses (you should see the input form to add a course) and click the bookmarklet.
Paste the list you coppied from Piazza into the input dialog and click OK.

You should now see the courses list populated with your courses from Piazza!
